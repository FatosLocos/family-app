Philips Hue koppelen

  1. Open in het Hue Developer Portal je gebruikersnaam rechtsboven, kies Remote Hue API appids en daarna Add new Remote Hue API app.
  2. Registreer als redirect-URL: https://app.ligtvoet.tech/instellingen/hue/callback/.
  3. Vul hier de client ID en client secret van die app in en kies Opslaan.
  4. Kies Koppel Philips Hue en rond de Hue-aanmelding af.
  5. Kies daarna Bridge koppelen, druk op de fysieke knop van de Bridge en kies Voltooien.
  6. De lampen verschijnen vervolgens onder Huis, met aan/uit en helderheid.

Client secret, OAuth-tokens en de Hue application key blijven versleuteld op de server en worden niet aan de browser teruggegeven.
